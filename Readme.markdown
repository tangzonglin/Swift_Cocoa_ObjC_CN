## 生成HTML文档

``` bash
# 安装工具
$ brew install node
$ npm install -g markdown-styles

# 生成文档
$ (可选) rm -rf doc # 如果当前文件夹存在doc文件夹，先删除它
$ generate-md --layout github --input . --output doc
```
